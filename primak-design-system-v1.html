01 — Logo

The logo is fixed. Everything else adapts.

The Primak wordmark is a geometric grotesque in bold/black weight, all lowercase. Rounded terminals on p, r, a provide warmth. The angular cut on the k provides edge. This tension — human within heavy — defines the brand's visual character.

Logo Anatomy

kAngular cutPrecision. Edge. Not purely soft.
ClassificationGeometric grotesque + humanist DNAFashion house, not tech startup

Logo Rules

  • Clear space: Minimum = height of the "p" on all sides
  • Minimum size: 80px width on screen, 20mm in print
  • Color: Black on white, white on black. No exceptions.
  • Backgrounds: Reads on #fff, #1a1a1a, #000 — all three
  • Never add effects (shadow, glow, gradient)
  • Never change proportions or weight
  • Never set in a different typeface
  • Never add a tagline lockup to the logo itself

05 — Visual Grammar

Structure is the decoration.

There is no decoration in PRIMAK's system. Structure — borders, grids, section numbers, rule lines — does all the visual work. This is not minimalism as aesthetic. It is minimalism as discipline.

Structural Elements

  • Rule lines: 3px solid #111 hero separator. 1px #e0e0e0 for grids.
  • Callout border: border-left 3px solid #111
  • Section numbers: 10px, uppercase, 3–4px tracking. Always first element.
  • Grid cells: border-collapse tables. Structure proves precision.
  • Large numerals: 48–56px in #ebebeb. Visual anchor without weight.
  • Section alternation: white → dark → black → white. Creates depth.
  • Dark bookends: Open dark, close dark. Every major document.
  • Proof bar: After hero. Uppercase, spaced, muted (#888).

Never Use

  • Gradients — any direction, any opacity
  • Box shadows — not even subtle ones
  • Rounded corners — flat edges only. 0px border-radius.
  • Accent colors — not even "subtle" ones
  • Decorative icons — or illustrations of any kind
  • Photography — in brand materials (B&W portrait is the one exception)
  • Animation — beyond simple hover state transitions
  • Cards with backgrounds — use borders instead
The Fashion House Principle: A fashion house doesn't produce a new style each season — it applies a consistent grammar across contexts. Chanel jacket, Chanel poster, Chanel perfume bottle — one grammar. When a client receives a proposal, sees the LinkedIn, opens the website — they feel the same house. Not "branded." An institution.
06 — Layout Patterns

Four document structures.

Pattern A: Dark Hero Document

CV, One-Pagers, major proposals. Opens with #111 hero (80vh), proof bar, white content sections, dark bookend close.

Pattern B: Section Alternation

Design briefs, research docs, long-form. White → dark → black → white rhythm. Each section = one topic.

Pattern C: Card Grid

Proof cases, service overview, comparison views. Cards with 1px #e0e0e0 border, no background. Hover: translateY(-2px). Tier 1 = full cards. Tier 2 = compact grid.

Pattern D: Clean Prose

Emails, outreach, internal docs. White background, max-width 680px, Inter 15px body, minimal structure. No hero. Footer only.
